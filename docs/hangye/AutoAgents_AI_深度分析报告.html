1.2 关键战略判断

2023 年“百模大战”时，杨劲松选择不训练大模型，理由有二，这两点构成了公司全部战略的起点：

判断一：大模型 = 基础设施

大模型会成为像水电一样的基础设施，但并非最终消费形态。自己训练模型是在和巨头拼资本，且无法形成壁垒。

判断二：企业要的是“能交付结果的系统”

企业需要的不仅是“聪明大脑”，而是一套能接入系统、调用工具、遵守权限、承担岗位任务的系统——这正是 Agent 要解决的问题。

不是单一产品，而是一套数字劳动力供应链

灵搭 Lingda 企业级 Agent 搭建平台 生产侧

面向业务人员的低代码智能体开发平台。提供 20+ 标准模块节点（用户提问、AI 对话、信息分类提取、知识库搜索、文档审查、图片识别、数据库查询等），支持 Text2Agent 自然语言生成工作流与 Skills 技能引擎。定位为“数字劳动力生产工厂”。

低代码20+ 模块节点Text2AgentSkills 引擎区别于 Coze/Dify 画布式产品，更面向非技术背景业务人员

袋袋 Profy 数字专家双边市场 流通侧

2026 年随 Pre-A 轮推出的战略新品，自我定位“数字劳动力的京东”。中小企业可直接采购/雇佣已训练好的智能体，按结果付费；领域专家则可把自身知识数字化为“数字分身”上架销售，获得持续收入。

双边市场按结果付费专家数字分身降低中小企业使用门槛，扩大 TAM

灵搭智能终端 桌面入口 / 岗位助手 消费侧

即将推出的桌面端产品，将智能对话、岗位专家、企业知识库、自动化任务和业务应用集中到统一入口，贴近一线工作流。本质是把 Agent 能力从“开发平台”下沉到员工日常工作面。

桌面入口岗位级知识库集成解决“建了没人用”的最后一公里

商业飞轮的关键不在三大产品本身，而在数据回流：智能体应用过程会沉淀任务规划路径、工具调用方式、问题解决过程、反馈结果等“操作轨迹数据”——这与传统大模型依赖问答数据截然不同。这些数据通过 RFT（强化学习微调）回流，让数字员工持续迭代，形成“用得越多 → 数据越多 → Agent 越强 → 替代更多岗位”的飞轮。

3技术架构深度剖析

五层模型：应用场景 → 产品矩阵 → Multi-Agent 引擎 → 大模型适配 → 云+端基础设施

前文已用分层架构图展示五层结构，这里聚焦最关键的技术创新点。

3.1 自研 Multi-Agent 架构：核心差异化

灵搭最核心的技术资产是其自研的“中心化调度 - 去中心化推理”框架，直接对标并试图解决传统 ReAct 架构的痛点。

传统 ReAct 架构的痛点

• 单 Agent 串行推理，复杂任务易触发上下文窗口限制

• 多步推理中错误累积，难以并行

• 单一 Agent 无法承担跨岗位的复合任务

灵搭的中心化调度+去中心化推理

• 协调智能体（Orchestrator）负责任务拆解与分发

• 专业智能体并行执行，各自维护独立上下文

• 知识工作流水线分工，提升一致性、降低错误

• 支持高并发任务处理与快速响应

注：该核心机制已申报多项发明专利，并作为首批通过中国信通院智能体标准认证的 Agent 平台。

5.1 标杆案例：电网数字员工的价值证明

以国家电网项目为例，最能体现“数字员工”而非“问答助手”的定位差异：

维度过去（资深专家）数字员工
变压器故障检测+运维鉴定报告约 1 周分钟级
合同审核效率—+80% 以上
覆盖岗位数—35 个数字岗位（供电服务/设备运维/合规审核/应急保障）
客户续约率—100%（20+ 电网客户）
场景策略洞察：未来式智能选择了一个非常聪明的切入点——电力行业对“稳定性、合规、可审计”的极端要求恰好放大了 Agent 相比纯大模型的优势（可调用工具、可约束权限、可留痕）。一旦在最高门槛行业立住，向金融、制造的复制就形成了“降维可信度”。这是典型的“先难后易”行业切入策略。

8挑战与风险

务实路线面临的现实压力

风险一：大模型能力跃迁可能压缩 Agent 中间层价值。若基座模型原生长上下文、原生工具调用、原生多步推理能力快速突破，“中心化调度-去中心化推理”这类自研编排层的差异化可能被稀释。灵搭需要持续证明其编排与 RFT 仍带来增益。
风险二：袋袋双边市场的冷启动难题。“数字劳动力的京东”需要同时有足够多的优质数字专家供给与足够多的中小企业需求，双边市场普遍面临鸡生蛋问题；专家“数字分身”上架的动力与分成模型尚待验证。
风险三：从大客户项目制到规模化产品的转化。当前收入高度依赖电网等头部客户的项目实施，2026 年 1 亿目标对团队规模（<50 人）和交付能力是考验。袋袋能否把“头部验证”转化为“中小企业自助使用”是规模化关键。
风险四：国资与产业资本进入后的战略节奏。政策红利带来资金也带来节奏要求（如 2027 年智能体普及率超 70% 目标），公司需要在“做深垂直”与“做广渠道”之间平衡，避免被资本催熟稀释护城河。
风险五：赛道拥挤与同质化。2025-2026 年 Agent 创投爆发，Coze、Dify、智谱、百度、阿里都在企业 Agent 方向加码，未来式智能的“业务人员低代码 + 高合规行业”定位虽错位，但窗口期有限。

9总结与展望

一家“把大模型变成数字员工”的中国样本

未来式智能（AutoAgents.ai）是国内 Agent 赛道中少有的“先难后易、先深后广”型公司。它的价值不在技术噱头，而在三件事的咬合：

① 选对了切入点
电力等高合规行业
用最严苛场景验证 Agent 可信执行
② 做对了架构
云+端 + Multi-Agent
同时解决合规、权限、稳定性
③ 找对了飞轮
操作轨迹数据 + RFT
区别于问答数据的独占复利资产

它的下一步看点集中在三个问题：袋袋市场能否跑通双边飞轮、灵搭终端能否成为员工日常入口、L3→L4 的跃迁是否如期。这三者若有两者跑通，未来式智能有机会从“Agent 工具供应商”升级为“数字劳动力基础设施”；若都受阻，则可能停留在“高续约的项目制 Agent 实施商”。

一句话总结：在所有人都在卷大模型的 2023 年，未来式智能选择做“让大模型会做事”的那一层；三年后，当模型格局已定、Agent 风口到来时，它已经站在了电力、能源等最高门槛行业的护城河里。它的命运，将取决于能否把“行业深度”转化为“市场广度”。
