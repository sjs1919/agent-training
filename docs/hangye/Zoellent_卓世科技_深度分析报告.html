三、产品体系全景

卓世科技构建了覆盖"行业模型→场景应用"的全栈产品体系，以MaaS（模型即服务）和智能体（Agent）双引擎为核心技术平台。完整产品矩阵如下：

3.1 璇玑玉衡大模型体系（技术底座）

  • 架构：MoE（混合专家）架构，千亿参数规模
  • 四大增强能力：知识增强、检索增强（RAG）、决策增强、跨模态增强
  • 技术优势：模型安全、模型压缩、边缘计算、RAG和Prompt压缩，在同等成本和效率下大幅提升模型性能和输出准确性
  • 垂直模型矩阵：工业大模型、康养大模型（全国首个养老行业千亿大模型）、出版大模型、教育大模型、中医大模型等
  • 合规资质：网信办首批大模型+深度合成算法双备案，海南自贸港首张也是唯一一张大模型牌照

3.2 三大核心平台（三位一体）

天璇 MaaS 大模型平台 — 行业模型"孵化器"

  • 提供企业、工业、康养、文教等垂直领域大模型的快速训练、部署与管理能力
  • 支持多源异构模型兼容，企业无需从零构建大模型
  • 低代码、一站式开发平台，降低行业模型构建技术门槛
  • 支持企业快速定制专属大模型应用

玉衡 Data 数据平台 — 数据智能"燃料库"

  • 打通数据治理、清洗、标注与知识沉淀全流程
  • 为大模型提供高质量、结构化的行业数据
  • 确保模型在复杂业务场景下的推理精度与决策可靠性
  • 在教育领域深耕四年积累的高质量语料成为训练行业模型的核心资产

天玑 Agent 智能体平台 — 场景落地"执行中枢"

  • 向上承接企业业务需求，向下连接行业大模型与终端设备
  • 构建从通用Agent到场景Agent的分层智能体体系

  • GCA（生成式对话智能体）：融合大模型+行业模型+RAG，提供端到端的对话式服务
  • GDA（生成式决策智能体）：融合大模型+行业模型+知识湖+智能体，通过对话方式构建用数流程、知识体系、经营决策体系与目标管控体系
  • 零代码编排，从单工作流到多智能体的柔性架构

三大平台协同联动，形成"数据→模型→智能体→业务"的全链路闭环。

四、Tri-Core 三核协同具身智能通用大脑

行业首个集成直觉（System 1）、逻辑（System 2）、共情（System 3）的"三核协同"具身大脑解决方案
在 SimplerEnv、LIBERO 等国际权威具身智能评测基准中取得 SOTA 水平，超过 π0、NVIDIA GROOT N1.5 等前沿模型。

架构设计：三核闭环

受认知心理学双系统理论启发，采用类脑模块化设计，区别于主流单一VLA（Vision-Language-Action）模型架构。核心优势在于异步计算：System 1在边缘侧高频运行保证实时性，System 2在云端低频运行保证逻辑正确性，System 3并行守护安全底线。

系统定位技术核心运行位置
System 1
（小脑）		 VLA反应式控制 Mixture-of-Transformers (MoT) 架构，统一视觉语言模型(VLM)与动作专家；引入Flow Matching（流匹配）生成范式，以更少推理步数生成高质量连续动作轨迹；两阶段训练：跨具身预训练+单一具身后训练 边缘侧高频
System 2
（大脑）		 VLM逻辑规划器 基于璇玑玉衡基座大模型构建；负责因果推演与复杂任务拆解；执行不可逆操作前强制进行隐空间模拟，基于物理常识预测后果；具备通用大模型所不具备的物理一致性推理能力 云端低频
System 3
（共情系统）		 情感计算与伦理安全 实时处理视觉（面部微表情、姿态）和音频（语调、语速）信号；计算用户情感状态向量影响System 2推理决策；充当伦理安全过滤器，拥有比System 2更高的中断权限，确保绝对安全 并行守护

System 1 关键技术解析

  • MoT架构：将视觉语言模型(VLM)和动作专家(Action Expert)统一建模，而非早期VLA模型（如RT-2）采用的离散化Token自回归生成
  • Flow Matching：不同于扩散模型需要繁琐的去噪步数，通过学习从高斯噪声到目标动作分布的向量场，以更少推理步数生成高质量连续动作轨迹，解决人型臂动作不平滑问题
  • 视觉专家：System 1的小尺寸VLM由System 2的大尺寸VLM通过知识蒸馏获得
  • 两阶段训练：Stage 1跨具身预训练（多种形态机器人数据，动作空间不一致但提升泛化认知能力）→ Stage 2单一具身后训练（场景精度优化）

五、NextClaw（龙虾）全栈具身智能体执行引擎

2026年3月9日发布 — 行业内少数真正打通IT云端与OT物理层的全栈执行引擎。不止是承载指令的软件平台，更是连接大模型大脑、底层硬件、存量系统与具身机器人的"神经中枢"。

行业痛点：大厂方案的四大盲区

盲区大厂方案问题实体场景影响
重IT轻OT只能对接现代化SaaS工业/康养场景老旧业务系统、传感器、非标硬件无法对接
重云端轻边缘数百万私有云交付，极度依赖网络工厂车间和养老院断网即瘫痪
高幻觉低容错通用千亿模型解析复杂工业图纸/医疗SOP易幻觉实体场景中是灾难性的
卖空壳难落地交付PaaS平台，需传统企业自己写Prompt/开发接口使用门槛极高

NextClaw 四大核心壁垒

壁垒一：下沉至OT物理层

深度集成ROS 2、Modbus、MQTT等底层工业与物联网协议。不仅能帮企业写报表，更能直接驱动物理世界：控制人形机器人动作执行、机械臂分拣、指挥AGV小车避障、响应护理床的物理报警。

壁垒二：边缘计算 + 断网可用

即便在断网离线环境下也能稳定运行，完美适配工厂车间、病房、户外作业等复杂实景。将AI推理能力下沉到边缘设备，摆脱对云端的绝对依赖。

壁垒三：反幻觉行业模型

基于璇玑玉衡行业大模型，深度融合行业知识、业务流程与实操逻辑，在解析复杂工业图纸和医疗SOP时大幅降低幻觉率。

壁垒四：落地交付非空壳

不交付PaaS平台让客户自己开发，而是提供完整的端到端解决方案。Agent X 2.0即将支持图形化拖拽生成数字员工，让业务人员"零代码"开发AI应用。开放100+个API接口，联合华为昇腾、海光、麒麟等国产软硬件伙伴。

七、核心差异化与竞争壁垒

7.1 四道护城河

  1. 行业大模型先发壁垒：2018年起深耕教育领域积累高质量语料，2022年战略定调不做通用模型专攻行业模型，比DeepSeek开源热潮早2年布局，连续两年行业模型市场份额第一梯队

  3. 具身智能OT下沉壁垒：Tri-Core三核协同大脑+NextClaw全栈执行引擎，深入ROS2/Modbus/MQTT底层协议，解决大厂不愿碰的"脏活累活"，与优必选战略合作构建"本体+大脑"生态
  4. 国家级合规与标准壁垒：网信办首批双备案、国家级专精特新重点"小巨人"、牵头制定工业智能体大模型标准、康养大模型行业标准，106项发明专利构建技术专利池

  • 具身智能商业化验证：Tri-Core和NextClaw发布仅数月，万台机器人部署计划尚处早期，实际落地效果待验证
  • 大模型厂商降维打击：百度/阿里/华为等大厂若推出行业模型+Agent平台一体化方案，将构成直接竞争
  • 康养领域天花板：养老行业数字化预算有限，G端客户付费能力和意愿可能成为增长瓶颈

十、总结与展望

核心价值主张

卓世科技的核心价值在于"从虚拟AI到物理AI的全链路打通"。当大多数Agent公司仍在IT云端编排API时，卓世科技已经构建了从行业大模型（璇玑玉衡）→智能体平台（天玑Agent）→具身智能大脑（Tri-Core三核协同）→OT执行引擎（NextClaw龙虾）的完整技术栈，将AI的能力延伸到了物理世界的最后一米。

发展路径总结

  1. 2018-2022 深耕积累期：百度系团队创业，教育领域语料积累4年，联想之星天使轮启动
  2. 2022-2024 战略定调期：不做通用模型专攻行业模型，迁址海南获取首张大模型牌照，构建三大平台全栈体系
  3. 2024-2025 规模化落地期：连续入选大模型产业图谱三大板块，7000+客户验证，康养行业标杆树立
  4. 2026 物理AI跃迁期：发布Tri-Core具身大脑+NextClaw执行引擎，引入优必选战略合作，Pre-IPO数亿融资，冲刺上市

关键竞争优势

  • "不做通用"的战略定力：2022年就判断通用大模型是基础设施，创业公司应深耕行业，比行业大模型热潮早2年布局
  • "数据→模型→智能体→业务"闭环：终端采集物理世界数据反哺模型，形成正向飞轮
  • 具身智能OT下沉能力：行业极少数能深入ROS2/Modbus/MQTT底层协议的AI公司，解决大厂不愿碰的"脏活累活"
  • "本体+大脑"生态绑定：与优必选科技战略合作，联合部署万台机器人，构建具身智能完整闭环
  • 合规先行：网信办首批双备案+国家级重点小巨人+多项行业标准牵头制定，构建政策护城河

未来关注点

  • IPO进程：Pre-IPO轮完成，关注上市时间表和估值表现
  • 万台机器人部署落地：与优必选合作的具身智能商业化验证
  • Agent X 2.0发布：图形化拖拽生成数字员工，降低使用门槛
  • 100+园区推广：国资园区复制效果
  • 行业模型市场份额：能否在DeepSeek开源冲击下保持第一梯队地位
