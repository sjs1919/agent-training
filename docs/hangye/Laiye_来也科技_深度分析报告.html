2.1 三大核心技能层（十年沉淀）

APA 智能体流程自动化 Agentic Process Automation 执行层

新一代智能体流程自动化产品，融合 LLM 能力。与传统 RPA 的根本区别：不依赖固定规则与结构化数据，能理解上下文、处理非结构化信息、自主决策。支持自然语言描述自动生成流程——业务人员用中文描述需求即可自动构建自动化流程，开发效率提升 10 倍。

低代码+自然语言UI自动化自适应修复解决传统 RPA 流程脆弱痛点

ADP 智能体文档处理 Agentic Document Processing 认知层

用智能体架构重新定义文档处理。核心技术：VLM 视觉理解版式 + LLM 语义理解内容 + 智能体编排多模型协同。零样本学习（无需标注训练）、100+ 语言支持、复杂版式（无框线表格/多栏/手写体/盖章遮挡）理解。实测文档解析 7,000 页/小时，海外发票 92.3% 准确率。

零样本100+语言人机协同数据飞轮OCR→IDP→ADP 范式跃迁

ACX 智能体客户体验 AI-Native Customer Experience 交互层

把"吾来"对话 AI 升级为 AI 原生客户体验平台。多模态意图触发、多轮对话、读写数据表、个性化问答；意图合并与重新聚类，知识库持续优化。把对话洞察转化为增长引擎。

Laiye Worker 桌面端 AI 员工（2026 全新发布） 消费入口

全生命周期管理调度监控审计多租户企业级治理

工作流编排（稳定）

基于企业确定的 SOP 构建，稳定、高效、可预测。适合规则明确的重复流程。

自主规划编排（灵活）

智能体根据上下文自主调用、组合工具，无需人工干预完成跨系统复杂流程。适合非结构化、动态任务。

企业可按场景灵活选择或组合使用，这是来也区别于纯 RPA（只有工作流）和纯 Agent（只有自主规划）的关键。

生态护城河，对标 UiPath 社区版战略

8挑战与风险

老牌 RPA 厂商转型 Agent 的现实压力

风险一：RPA 历史包袱与 Agent 范式切换的张力。来也的核心收入仍来自 RPA 项目与平台授权。APA 虽然升级了 RPA，但客户认知仍停留在"自动化工具"，向"自主 Agent"的心智迁移需要时间。若大模型原生 Agent 能力快速突破，RPA 式的"录屏+规则"范式可能被颠覆。
风险二：ADP 在 IDP 魔力象限中的位置。来也被 Gartner 定位为"特定领域者"（非领导者象限），意味着在市场份额与全球化方面仍落后于 ABBYY、UiPath 等国际巨头。零样本+智能体架构虽是技术亮点，但商业化规模仍需验证。
风险三：Laiye Worker 的规模化与生态冷启动。四档定价试图打开长尾市场，但个人 AI 员工赛道竞争激烈（与各类 AI 助手/Agent 产品同台），社区版积分制能否真正转化为付费订阅尚待观察。开发者社区 80 万是 RPA 时代积累，能否迁移到 APA 生态是关键。
风险四：大模型厂商自带 Agent 的挤压。OpenAI、智谱、百度等大模型厂商正把 Agent 能力内置到模型中，可能压缩"中间编排层"的价值。来也的护城河在于"执行层"（真能操作系统），但若模型原生具备 GUI 操作能力，这一层也会被稀释。
风险五：国际化与国内市场的平衡。来也在新加坡、英国设总部，收购法国 Mindsay，但主要收入与案例仍来自国内。Gartner 象限的认可不等于海外市场份额，与 UiPath、Automation Anywhere 的全球竞争仍需持续投入。

9与未来式智能对比：两种 Agent 路径

同为"数字员工"，截然不同的起点与路径

维度来也科技未来式智能

路径差异的本质：来也走的是"自下而上"路径——先有扎实的执行能力（RPA），再叠加认知（ADP）与交互（ACX），最后用 Agent 重新封装；未来式智能走的是"自上而下"路径——先有 Multi-Agent 编排与行业 Know-How，再向下补执行能力。前者护城河在"能做事"，后者护城河在"懂行业"。两者在电力行业交汇，但客户画像与商业阶段不同。

10总结与展望

一家用十年积累卡位智能体时代的"老树新枝"

来也科技是国内智能自动化赛道少有的"全栈型"公司——同时具备执行层（APA/RPA）、认知层（ADP）、交互层（ACX）、平台层（WEP）与入口层（Worker）的完整能力栈。它的价值不在单一技术突破，而在四件事的咬合：

① 选对了赛道演进
RPA→AI Agent
叠加进化而非推倒重来
② 做对了技术路径
ADP 智能体化
用 Agent 架构重定义文档处理
③ 卡对了合规位置
信创+三象限
高合规行业准入证
④ 找对了普惠路径
Worker+社区版
把企业级能力下沉长尾

它的下一步看点集中在三个问题：Laiye Worker 能否打开个人/中小企业订阅市场、ADP 能否从 IDP"特定领域者"跃迁到领导者象限、APA 能否成为智能体流程自动化的事实标准。若三者有一者跑通，来也将从"智能自动化厂商"升级为"数字员工基础设施"；若都受阻，则可能在 Agent 范式下被更原生的创业公司与大模型厂商夹击。

一句话总结：来也科技是 Agent 浪潮中"底盘最稳"的玩家——十年 RPA 执行能力 + 三大魔力象限认证 + 全栈信创合规，让它在大客户市场拥有纯 Agent 创业公司难以速成的护城河。它的命运，将取决于能否把"企业级执行能力"成功封装为"普惠化的 AI 员工"，从项目制收入平滑过渡到订阅制规模。
