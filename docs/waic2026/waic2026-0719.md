- **产业链分化**：从整机向"大脑（大模型）→ 小脑（运控）→ 灵巧手 → 关节 → 数据"全链条细分
- **上市潮将至**：宇树科技已注册生效，乐聚智能 IPO 已受理，跨维智能拟启动 IPO，云深处启动科创板 IPO

### 3.3 产业链成熟度评估

| 阶段 | 赛道 |
|------|------|
| **成熟期**（已上市/规模化） | 智能语音（科大讯飞）、工业自动化（西门子/ABB）、企业软件 |

| **爆发期**（融资火热，估值高，未盈利） | 具身智能、大模型、脑机接口、量子计算 |
| **早期**（方向探索中） | AI Agent、AI 视频生成、AI 3D/空间 |

### 3.4 值得关注的冷门高潜力赛道

1. **触觉传感器/电子皮肤**：从科研走向产业化，他山科技、纬钛科技等融资加速

5. **量子计算**：图灵量子估值近 70 亿，不筹量子天使轮即数千万
