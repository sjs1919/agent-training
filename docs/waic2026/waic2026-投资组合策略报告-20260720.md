## 一、核心摘要

### 1.1 WAIC 2026 揭示的产业趋势

WAIC 2026 参会企业 963 家的数据结构，清晰指向 AI 产业链正经历从"模型竞赛"向"算力基础设施+具身智能硬件"的双主线迁移：

- **算力·芯片·基础设施**（226家，23.5%）— 国产 GPU 进入集中上市期，AI 芯片从"可用"走向"好用"
- **机器人与智能硬件**（233家，24.2%）— 具身智能单笔融资频创新高，产业链从整机向零部件全链条分化
- **大模型格局已定**（仅20家做基础大模型）— 头部集中度极高，应用层AI Agent（87家）百花齐放

### 1.2 投资结论（一句话）

**当前AI板块7月回调是拥挤度消化+业绩验证期的正常调整，不是趋势逆转。建议维持AI算力核心仓位，优化结构：减持纯光模块（新易盛），增配先进封装（长电科技）+ DDR5互连（澜起科技），保留端侧AI（瑞芯微）和半导体设备（微导纳米）作为成长弹性仓位。**

## 三、供应链价值量深度拆解

### 3.1 AI 算力产业链价值量分布

以单台 AI 服务器（8×GPU）为基准的价值量拆解：

```
AI服务器总成本 ≈ $250-300万（8×GPU配置）

GPU加速卡        ~$160-200万   65-70%   ← 价值量最大，但GPU芯片被英伟达垄断
HBM高带宽存储    ~$25-30万     10-12%   ← 澜起科技（内存接口/RCD芯片）
光模块(800G×8)   ~$8-12万       3-4%    ← 中际旭创、新易盛
PCB/载板         ~$5-8万        2-3%
先进封装(CoWoS)  ~$15-20万      6-8%    ← 长电科技（国内先进封装替代）
服务器/散热      ~$20-25万      8-10%
其他             ~$5-8万        2-3%
```

**国产替代增量空间测算**：

| 环节 | 当前国产化率 | 2028E国产化率 | 增量空间（亿元） | 受益标的 |
|------|------------|-------------|----------------|---------|
| AI GPU | <5% | 15-20% | 500-800 | 海光信息、寒武纪 |
| 先进封装 | ~15% | 35-40% | 300-500 | **长电科技**、通富微电 |
| 光模块 | ~40% | 60-70% | 200-300 | 中际旭创、新易盛 |
| DDR5内存接口 | ~30% | 55-65% | 150-250 | **澜起科技** |
| 半导体设备 | ~20% | 35-40% | 500-800 | 北方华创、**微导纳米** |

### 3.2 具身智能产业链价值量分布

以单台人形机器人（量产成本目标 ~15-25万元）为基准：

```
人形机器人总成本 ≈ 15-25万元

关节模组(28-40个)    ~6-10万    40%     ← 绿的谐波（减速器）、汇川技术（伺服）
丝杠/传动系统         ~2-4万     13-15%  ← 微导纳米（精密制造相关）
传感器(力/触觉/视觉)  ~2-4万     13-15%  ← 柯力传感、敏芯股份
AI芯片/计算平台       ~1.5-3万   10-12%  ← 瑞芯微（端侧AI SoC）
电池/电源             ~0.8-1.5万  5-7%
结构件/其他           ~2-4万     13-15%
```

### 3.3 国产替代的逻辑叠加效应

每家标的的**总价值量 = 供应链自然增长 × 国产替代倍率**：

| 标的 | 供应链自然增长(CAGR) | 国产替代倍率 | 综合增长(CAGR) | 关键驱动 |
|------|---------------------|-------------|---------------|---------|
| 长电科技 | 15-20% | 2.0-2.5x | **30-45%** | CoWoS国产化+先进封装渗透率 |
| 澜起科技 | 20-25% | 1.8-2.2x | **35-50%** | DDR5渗透+DDR6预研+PCIe Retimer |
| 新易盛 | 25-30% | 1.2-1.5x | **30-40%** | 800G→1.6T升级周期 |
| 瑞芯微 | 20-25% | 2.5-3.0x | **50-70%** | 端侧AI从0到1，替代高通/MTK |
| 微导纳米 | 15-20% | 3.0-4.0x | **45-65%** | ALD设备从几乎0%国产化起步 |

### 4.3 微导纳米 (688147) — 当前持仓

#### 核心逻辑
- **ALD（原子层沉积）设备国产龙头**，技术壁垒极高
- 存储芯片扩产（长江存储/长鑫存储）直接受益
- ALD在先进制程（<7nm）中不可替代，国产化率几乎从0起步
- 半导体设备国产替代政策最强催化

#### 近期表现
- 7月第二周（7.6-7.10）涨7.72%
- 但估值约413倍，市场给予极高成长溢价

#### 风险点
- 估值极高（~413x PE），波动剧烈
- 光伏设备业务阶段性承压（光伏行业产能过剩）
- ALD设备验证周期长，业绩释放慢于订单
- 流通市值较小，流动性风险

#### 操作建议
**继续持有但控制仓位（不超过15%）**。半导体设备是国产替代最确定的赛道，但微导纳米估值压力大，建议作为弹性仓位而非核心仓位持有。

## 五、2026年7月市场走势研判

### 5.1 AI/半导体板块整体走势

**7月以来板块特征**：

| 特征 | 具体情况 |
|------|---------|
| 指数表现 | 科创50/AI板块7月以来回调5-10% |
| 回调原因 | ① 前期涨幅过大，拥挤度偏高；② 中报业绩验证期资金趋于谨慎；③ 部分资金高低切换 |

| 北向资金 | 阶段性净流出科技板块 |

**机构核心观点汇总**：

- **华泰证券**："短期科技超跌后或有修复，但拥挤度压力充分消化前可能仍处于高波动状态。科技内部建议规避纯题材和高拥挤方向。"
- **券商晨会共识**："等待AI资本开支和业绩验证，规避纯题材和高拥挤方向"
- **机构普遍判断**：AI与半导体长期成长逻辑未改，短期承压不改变景气持续上行

## 六、调仓建议与投资组合推荐

### 6.1 调仓逻辑

**从新易盛切换到长电科技+澜起科技的核心逻辑**：

1. **产业链位置迁移**：从"光模块"（偏中游制造，周期性较强）→"先进封装+DDR5"（偏上游核心环节，壁垒更高）
2. **国产替代弹性升级**：光模块国产化率已~40%，继续提升空间有限；先进封装国产化率仅~15%，DDR5接口国产化率~30%，替代空间更大
3. **估值性价比**：新易盛6700亿市值已充分定价，长电和澜起仍有估值提升空间
4. **风险分散**：光模块过度依赖海外云厂商CAPEX，先进封装+DDR5受益于更广泛的AI算力需求

### 6.2 "微微套住"的处理策略

> **当前持仓处于微亏状态，不建议在亏损时恐慌性割肉。**

具体策略：
- **新易盛**：利用中报预增利好（利润+77~103%）带来的反弹窗口减仓，不必在低点割肉
- **瑞芯微**：中报同样超预期（+60~71%），端侧AI逻辑强化，解套概率高
- **微导纳米**：半导体设备是政策最确定的方向，存储扩产+ALD国产替代逻辑未变，耐心持有

### 7.4 仓位管理原则

1. **总仓位控制在70-85%**，保留15-30%现金应对波动
2. **单一标的不超过25%**，避免集中度风险
3. **分批建仓**，不在单一点位全仓买入
4. **设置止损线**：单票亏损超过15%强制审视逻辑是否破坏
5. **不追高**：若标的短期涨幅>30%，暂停加仓，等待回调
