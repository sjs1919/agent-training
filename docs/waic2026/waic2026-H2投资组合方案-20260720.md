## 一、顶层政策定调：习近平 WAIC 2026 主旨讲话解读

### 1.1 历史性信号

**这是习近平首次出席世界人工智能大会并发表主旨讲话。** 在中美科技博弈深水区，这一动作将 AI 产业提升至国家最高战略层级，信号意义远超历届 WAIC。

### 1.2 讲话核心框架（四点核心方向）

| 方向 | 核心表述 | 产业映射 | 受益环节 |
|------|---------|---------|---------|
| **自立自强** | "坚持自立自强，突破关键核心技术" | 国产芯片/设备/基础软件替代 | AI芯片、先进封装、半导体设备、DDR5 |
| **突出应用** | "打通科技加速向现实生产力转化通道" | AI从"实验室"走向"产线" | 具身智能、AI+制造、端侧AI |
| **安全治理** | "强化风险意识，反对泛化国家安全" | AI安全标准、可控生态 | 国产算力基础设施、数据安全 |
| **开放合作** | "提供5000个AI培训名额，全球AI治理框架" | 中国AI技术输出 | 大模型出海、标准制定 |

### 1.3 与 2026 下半年投资最相关的三句话

```
① "坚持自立自强" → 国产替代是最硬的逻辑线
② "突出应用导向" → 纯讲故事不行了，必须有业绩兑现
③ "打通科技→生产力通道" → 设备/制造环节比纯算法更受益
```

**结论**：2026H2 的主线不是"AI 概念炒作"，而是**"国产替代落地 + 业绩验证"**。习主席亲自站台 WAIC，意味着政策催化已到顶点，接下来要看真金白银的业绩。

## 二、WAIC 2026 产业格局：从 963 家展商看资金该去哪

### 2.1 赛道筛选逻辑

```
过滤器 1：WAIC 展区规模大（赛道热度高）        → 具身智能(233家)、算力芯片(226家)
过滤器 2：有实质融资/上市数据（产业成熟度）    → AI芯片/半导体设备/先进封装
过滤器 3：国产化率 < 30%（替代空间大）         → 先进封装(15%)、ALD设备(5%)、DDR5(30%)
```
