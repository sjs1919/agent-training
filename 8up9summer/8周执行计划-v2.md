# 八升九暑假执行计划 · v2

> 制定日期：2026-07-10
> 适用学生：深圳初二升初三
> 周期：2026.7.13 - 2026.8.31（共 7 周 + 2 天）
> 学习日：周一至周五学习，周六上午休息/下午化学，周日白天休息/晚上预备
> 模式：A日（英语专项 + 数学）/ B日（语文阅读 + 物理）
> 来源：[AI:Claude]

## 五、阶段安排

### 第一阶段：基础复盘期（7.13 - 8.9，4 周）

**主题**：巩固 7-8 年级基础，建立每日学习习惯。

| 科目 | 任务 |
|------|------|
| 英语 | 7-8 年级 32 单元核心词+短语，每天 1-2 单元；完形+阅读基础训练；语法漏洞补漏 |
| 语文 | 7-8 年级中考高频古诗文，每天 1-2 首古诗或 1 篇文言文；阅读模板积累 |
| 数学 | 七上→八下课本回归，每周一册，只刷基础题和易错题；每周 1 次限时计算训练 |
| 物理 | 八上+八下力学补漏，基础概念+基础计算；每周 1 次限时训练 |
| 化学 | 基础期不安排，保证主科和休息 |
| 历史/道法 | 隔天 20min 阅读，单日历史、双日道法 |
| 体育 | 每天 ≥30min 运动，以放松活动为主 |

### 第二阶段：过渡提升期（8.10 - 8.23，2 周）· 强度提升

**主题**：适度九上预习 + 中考体育专项启动，整体强度比第一阶段提升一档。

| 科目 | 任务 |
|------|------|
| 英语 | 9 上 8 单元复盘（单词+课文+语法）；新中考题型专项；每周 2 篇作文 |
| 语文 | 9 上古诗背诵默写 + 文言文预习 + 作文升格；名著阅读推进 |
| 数学 | 9 上第 1-2 章预习；每周接触 1-2 道选填压轴或几何综合题 |
| 物理 | 9 上电学入门（电流、电压、电阻、欧姆定律） |
| 化学 | 提升期安排 1-2 次（周日下午），九上第 1-2 单元；基础期不安排 |
| 历史/道法 | 隔天 20min 保温阅读 |
| 体育 | 周一/三：4 分钟跳绳 + 1 分钟速跳保持；周五：1-2 公里跑；周六/日：排球专项 |

### 第三阶段：收心准备期（8.24 - 8.31，8 天）

| 日期 | 重点 |
|------|------|
| 8.24-8.26 | 7-8 年级易错点、错词本、错题本快速回顾；化学课本简单翻阅 |
| 8.27-8.29 | 9 上英语/语文重点再过一遍，作息按开学时间调整；化学知识点回顾 |
| 8.30 | 体育模拟测试 + 实验操作视频学习 |
| 8.31 | 准备文具资料、整理书包、暑假学习总结 |

### 化学（P3 · 提升期周日下午）

**调整说明**：
- 基础复盘期不安排化学，保证主科和周六全天休息；
- 过渡提升期化学安排在周日下午，但需注意提升期只有两个周日（8.16 和 8.23），其中 8.23 有过渡提升测；
- 因此建议提升期安排 **1 次化学课（8.16 周日下午）**，另一次可灵活安排在工作日晚上或收心期简单阅读时补充。

| 次数 | 时间 | 内容 |
|------|------|------|
| 第 1 次 | 8.16 周日下午 | 化学入门 + 物质的变化和性质 + 我们周围的空气 |
| 补充 | 8.24-8.31 收心期 | 化学课本简单翻阅 + 知识点回顾 |

**注意**：如果希望提升期上 2 次化学课，第二次只能放在 8.15 周六下午或工作日晚上，会占用周六整块休息时间。

**目标**：暑假结束时对化学有基本认知，开学后能跟上课堂节奏。

## 九、执行纪律

1. **背诵**：动口 + 动笔，每天最后 5 分钟默写检测。
2. **限时**：数理基础题限时训练，模拟考试节奏。
3. **错题**：当天整理，每周五集中复盘，同类错误连续做对 3 次过关。
4. **周末**：周六全天休息 + 晚上自由活动；周日白天休息、提升期下午化学、晚上预备。
5. **运动**：基础期每天 ≥30min 放松运动；8.10 起周一/三晚饭前跳绳、周五跑步、周末排球。
6. **英语不断档**：A日专项、B日错词/语法、周六听力/阅读保温、周日错词复习。
7. **打卡**：每天按《每日打卡表-v2》勾选完成项。
8. **简报**：每周五输出一页学习简报，家长签字。
9. **留白**：相信计划表上的休息时间，家长不在孩子休息时催促。
