# 八升九暑假执行计划

> 制定日期：2026-07-10
> 适用学生：深圳初二升初三
> 周期：2026.7.13 - 2026.8.31（共 7 周 + 2 天）
> 学习日：周一至周五，周末休息，8.24-8.31 为收心期
> 模式：A日（英语专项 + 数学保温）/ B日（语文阅读 + 物理/化学）隔日交替
> 来源：[AI:Claude]

---

## 一、核心策略

基于学生画像，暑期优先级如下：

| 优先级 | 科目 | 策略 |
|--------|------|------|
| **P0** | 英语 | 唯一真短板，每天专项突破（单词 + 语法 + 阅读 + 写作） |
| **P1** | 数学、物理 | 理科强项，以保温和基础复盘为主，保持手感 |
| **P2** | 语文 | 冲 A+，重点抓阅读模板、古诗文背诵、作文 |
| **P3** | 历史、道法、化学 | 历史保温、道法补漏、化学入门 |

## 三、A日 / B日 轮换规则

| 类型 | 日期 | 上午 10:25-12:00 | 下午 14:30-16:00 | 下午 16:20-17:30 |
|------|------|------------------|------------------|------------------|
| **A日** | 周一/三/五 | 英语专项（语法/完形/新中考阅读） | 数学保温 | 英语错题 / 写作素材 |
| **B日** | 周二/四 | 语文阅读（现代文/文言文） | 物理 / 化学 | 语文积累 / 作文 |

**周排示例**：
- 第 1 周：一(A) 二(B) 三(A) 四(B) 五(A) 六休 日休
- 第 2 周：一(A) 二(B) 三(A) 四(B) 五(A) 六休 日休
- 以此类推

## 四、阶段安排

### 第一阶段：基础复盘期（7.13 - 8.9，4 周）

**主题**：巩固 7-8 年级基础知识，建立每日学习习惯。

| 科目 | 任务 |
|------|------|
| 英语 | 7-8 年级 32 个单元核心词+短语滚动，每天 1-2 单元；完形+阅读基础训练 |
| 语文 | 7-8 年级必背古诗文系统过一遍，每天 1-2 首古诗或 1 篇文言文 |
| 数学 | 七上→八下课本回归，每周一册，只刷基础题和易错题 |
| 物理 | 八上+八下力学补漏，基础概念+基础计算 |
| 化学 | 第 2/4 周各 1 次入门，九上第 1-2 单元 |
| 历史/道法 | 隔天 20min 阅读，单日历史、双日道法 |
| 体育 | 每天 ≥30min 运动，以放松活动为主 |

### 第二阶段：过渡提升期（8.10 - 8.23，2 周）

**主题**：适度九上预习 + 中考体育专项启动。

| 科目 | 任务 |
|------|------|
| 英语 | 9 上 8 个单元复盘（单词+课文+语法），新中考题型专项 |
| 语文 | 9 上古诗背诵默写 + 文言文预习 + 作文升格 |
| 数学 | 9 上第 1-2 章预习（一元二次方程、二次函数基础） |
| 物理 | 9 上电学入门（电流、电压、电阻、欧姆定律） |
| 化学 | 第 6 周 1 次，九上第 3 单元 |
| 历史/道法 | 隔天 20min 保温阅读 |
| 体育 | 每周 3-4 次中考体育专项训练，安排在周中晚饭前运动时间 |

### 第三阶段：收心准备期（8.24 - 8.31，8 天）

**主题**：开学准备 + 作息调整 + 错题滚动。

| 日期 | 重点 |
|------|------|
| 8.24-8.26 | 7-8 年级易错点、错词本、错题本快速回顾 |
| 8.27-8.29 | 9 上英语/语文重点再过一遍，作息按开学时间调整 |
| 8.30 | 体育模拟测试（中考项目） |
| 8.31 | 准备文具资料、整理书包、暑假学习总结 |

|------|------|------|

### 建议购置资料

| 优先级 | 科目 | 资料 | 作用 | 建议 |
|--------|------|------|------|------|
| P0 | 英语 | **新中考阅读**（老师推荐） | 深圳新题型阅读专项 | 必买 |
| P0 | 语文 | **万维阅读**（老师推荐） | 现代文/文言文阅读模板 | 必买 |
| P0 | 数学 | **《万唯中考基础题》数学** | 按知识点分类、基础题+少量中考真题 | 必买 |
| P1 | 英语 | 春如金卷 160篇 / 深圳中考英语语法填空 | 完形+阅读+语填训练 | 预算够再买 |
| P1 | 物理 | 《五年中考三年模拟》深圳专版 物理 | 知识点+深圳中考真题 | 必刷题做完/不够再买 |
| P2 | 物理 | 《名师金典·直击中考》 | 深圳本土资料 | 有余力再补充 |

**购买原则**：
1. 每科 1-2 本足够，不要堆资料；
2. 优先做完手头已有的空白题；
3. 深圳中考优先选择深圳专版或广东专版；
4. 数学是最需要补资料的科目，因为需要按 7-9 上知识点系统训练基础题。

## 八、执行纪律

1. **背诵**：动口 + 动笔，每天最后 5 分钟默写检测。
2. **限时**：数理基础题限时训练，模拟考试节奏。
3. **错题**：当天整理，每周五集中复盘，同类错误连续做对 3 次过关。
4. **周末**：周六全天休息，周日 19:00-20:00 做次日预备（含英语错词 30min）。
5. **运动**：基础期每天 ≥30 分钟；8.10 起每周 3-4 次中考体育专项训练，安排在周中晚饭前。
6. **英语不断档**：A日专项、B日错词复习、周六听力/阅读保温、周日错词复习。
7. **打卡**：每天按《每日打卡表》勾选完成项。
8. **简报**：每周五输出一页学习简报，家长签字。
