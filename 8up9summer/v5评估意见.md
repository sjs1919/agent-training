### 合理之处

3. **每日作息张弛有度**：9:00-22:00 内学习、休息、运动、睡眠边界清楚；22:00 睡觉保障 8 小时睡眠，符合初三暑假节奏。

5. **验收体系完整**：每日验收 + 阶段验收 + 每周五学习简报，家长、孩子、AI 工具三方都有抓手。

2. **家长操作负担仍偏重**：打卡表增加了大量勾选项（数学页码/题号、物理画图/规范计算、化学方程式书写、历史时间轴、道法索引等），家长需要熟悉多本书籍和验收话术，前两周容易手忙脚乱。
3. **缺少心理建设与亲子沟通节点**：计划全是学习任务，未明确「遇到困难怎么办」「如何反馈情绪」。
4. **升学规划未涉及**：四大自主招生、指标生政策、目标学校关注等未进入暑假视野。

### 班主任建议

- **立即修正周日冲突**：将晚饭+运动调整为 17:30-19:00，19:00-20:30 为晚间预备整块时间，正好容纳语文作文修改（30min）+ 英语错词/数学错题（30min）+ 化学方程式（10min）+ 名著/道法（20min）。

- **每周日晚增加 10-15min 家庭简短会议**：不问「完成了吗」，问「今天最难的是哪一科」「下周想调整什么」。
- **8 月起关注目标学校官网/招生群**，为指标生和自主招生预留信息窗口。

## 二、语文教师视角

### 合理之处

1. 古诗文背诵篇目覆盖 7-9 年级中考高频内容，与深圳中考默写 8-10 分考点匹配。
2. **B 日阅读四步法落地**：「限时做 → 对答案 → 总结模板 → 默写 2 句」将 75 分钟切为可验收的闭环，避免只做题不总结。
3. **作文形成闭环**：周四写 45min + 周日改 30min + 找出 3 个提升点，避免「只写不改」。
4. **名著扩面**：周六晚《水浒传》/《儒林外史》轮读 + 周日晚名著考点速览，覆盖深圳中考名著 6-8 分考查面。
5. 语文任务在打卡表中拆分到最小可勾选单元，家长可逐项检查。

### 不合理 / 需关注之处

1. **部分日期古诗/文言文篇目偏多**：如 7.15 古诗⑤⑥⑦ 三首、7.31《三峡》精读 + 古诗⑨⑩，30min 内背诵 + 重点字词 + 默写压力较大；若孩子记忆速度一般，容易只背不默、只记不牢。
2. **作文训练频次仍偏低**：暑假 7 周完整作文仅 7 篇左右，对「冲 A+」目标略显不足；周日修改本质上不是新写作。
3. **名著 30min/周可能不够**：《儒林外史》人物众多、讽刺笔法需要一定阅读量，30min 轮读可能只能蜻蜓点水。
4. **阅读模板总结依赖孩子自觉性**：四步法中「总结模板」若孩子不会提炼，容易流于抄答案；家长未必能判断模板质量。

### 语文教师建议

- **古诗日如遇篇目偏多，允许拆分到两天**：默写检测优先保证 2 句准确，不追求当天全部背完。
- **每周增加 1 次作文提纲/片段训练**：例如周二 16:45-17:30 素材积累后，用 10min 写一个开头或一个细节段落，提升写作熟练度。
- **名著阅读配合考点清单**：每周六晚阅读前，先给 3-5 个考点问题（如「林冲性格变化的关键事件」「范进中举的讽刺手法」），让阅读更有方向。
- **提供 5-8 个阅读答题模板示例**（环境描写作用、人物形象分析、句子含义理解、文言实词推断），降低孩子总结难度。

## 三、数学教师视角

### 合理之处

1. A 日下午数学独占约 2h，连续时间充足，适合深度思考。
2. **周五限时计算固定到 15:35-15:50**，与周复盘、错题整理形成周五闭环。
3. **课本回归只做概念+例题，主刷题用《万唯中考基础题》**，主次清晰，避免贪多嚼不烂。
4. **过渡提升期每周五 1 道深圳特色题**（新定义/综合与实践/尺规作图轮换），让学生提前见深圳中考风格。
5. 打卡表增加「完成页码/题号/限时/错题数/家长抽查题号」留白，验收颗粒度足够。

### 不合理 / 需关注之处

1. **「七上至八下每周一册」节奏偏快**：每册实际只有 1-2 天，对 90 分左右、选填容易失误的学生，基础复盘期不宜求快。
2. **深圳特色题 25min 可能偏紧**：新定义或综合与实践题审题、尝试、复盘往往需要 30-40min，周五 15:35-16:00 可能只够完成、不够消化。
3. **基础复盘期完全跳过压轴**：虽然计划定位保守，但学生数学为 A+、年级第 12，暑假完全不接触压轴，初三开学后面对二次函数、圆、相似综合可能手感生疏。
4. **家长抽查需要家长熟悉题目**：若家长数学基础较弱，抽查「讲思路和易错点」可能难以判断对错。

### 数学教师建议

- **每章最小闭环改为「概念 + 例题 + 5 道基础题」**，不要求每章全部刷完；耗时过长的章节可顺延到收心期。
- **特色题时间安排更灵活**：若 25min 做不完，可占用当周周五 20:00-20:30 自由阅读时段的 15min 进行复盘，或放到周日下午。
- **过渡提升期每周增加 1 道选填压轴或几何综合题尝试**：目标不是全对，而是熟悉中考压轴难度和常见模型。
- **为家长准备「抽查标准答案 + 关键步骤」**：让孩子口述时对照，降低家长判断难度。

## 四、英语教师视角

### 合理之处

1. **英语 P0 真正落地**：A 日约 2.75h + B 日 50min + 周日 15min 错词，周平均约 2.1h/天，与短板地位匹配。
2. **E 听说 20min 拆分具体**：模仿朗读 5min + 听力精听 10min + 信息转述 5min，可操作、可验收。
3. **B 日错词复习 + 语法专项闭环**：通过替换任务内容解决语法漏洞，避免 A 日新学、B 日不复习。
4. **深圳新题型专项落地**：8.12/8.19 两个周三上午替换为信息还原/语篇填空，覆盖深圳新中考能力点。
5. 作文批改流程「孩子自改 → 豆包改语法 → 家长看逻辑」清晰，降低家长英语批改负担。

### 不合理 / 需关注之处

1. **B 日英语时间偏少**：B 日仅有 9:40-10:10 单词/语法 + 10:15-10:35 E 听说共 50min，对英语 22.3 分的短板学生，隔天只有 50min 输入，保温作用有限。
2. **语法专项融入错词复习可能不够**：30min 内要复习错词又要补时态/语态/从句/非谓语，对基础薄弱学生可能捉襟见肘。
3. **英语作文仅 1 篇/周**：虽然目标已统一为 1 篇，但对 P0 短板科目，写作输出仍偏少。
4. **新题型专项仅 2 次**：信息还原、语篇填空各 1 次，开学前接触量偏少。

### 英语教师建议

- **B 日晚间可增加 10-15min 英语听力/错词**：例如 19:50 化学方程式之后，用 10min 听 1 段英语听力或复习 5 个错词，确保英语「不断档」。
- **语法错误率高的专题，在周日下午错词复习中滚动**：周日 19:30-19:45 英语错词可扩展为 20min，专门滚动本周高频语法错题。
- **每周五作文后，周日修改时要求孩子朗读自己的作文并录音自查**，强化语感和连接词使用。
- **新题型专项如时间允许，可在 8.24-8.31 收心期再各加 1 篇**，巩固手感。

## 五、物理教师视角

### 合理之处

1. B 日下午物理独占 14:30-16:00（化学启动前），适合力学/电学体系化学习。
2. **八下力学补漏 → 九上电学预习顺序合理**，符合深圳中考物理复习路径。
3. **每周 1 次限时训练、计算题「已知→求→解→答」四步书写**，对应深圳中考步骤分要求。
4. **实验视频提前嵌入 8.10-8.23**：8.13 欧姆定律实验、8.20 测小灯泡电功率，避免只到 8.30 才临时抱佛脚。
5. 打卡表物理任务增加「章节/实验/模型」「画图数量」「规范计算道数」「口述模型/实验」，家长可逐项抽查。

### 不合理 / 需关注之处

1. **理科连续 2h 对八升九学生仍偏长**：虽然有 5min 休息，但 15:35-16:30 第二小时效率可能下降，尤其电学入门阶段。
2. **实验视频仅 15-20min，且嵌入在刷题时段内**：若视频质量一般或孩子需要暂停记笔记，时间可能不够。
3. **物理计算题格式要求需家长能判断**：部分家长可能不熟悉「已知→求→解→答」的规范，抽查时难以把握尺度。
4. **八上声光热、八下力学、九上电学三周压缩**：第 1-3 周各覆盖一大模块，对力学薄弱点可能扫不透。

### 物理教师建议

- **允许在 15:30-15:35 休息之外，16:00 化学开始前增加 1-2min 站立活动**，缓解第二小时疲劳。
- **实验视频后强制口述 2min 要点**：实验目的、关键步骤、易错点，家长只听孩子讲，降低专业判断门槛。
- **为家长准备「计算题四步书写示例」**，抽查时对比即可。
- **力学薄弱章节（如浮力、简单机械）可适当放慢**，不要为赶进度牺牲理解。

## 六、化学教师视角

### 合理之处

1. **基础期（7.13-8.9）完全不安排化学**，保证主科和周六休息，符合零基础入门规律。
2. **化学方程式背诵固定到 19:50-20:00**，并有家长抽写 1 个完整方程式的验收，避免模糊执行。
3. **8.10 起每周二、四下午各 30min 入门**，循序渐进，不冲击主科。
4. **实验视频提前到 8.11/8.18**：氧气制取、二氧化碳制取/酸碱中和，覆盖深圳中考化学实验高频点。
5. 收心期有化学课本翻阅和知识点回顾，防止遗忘。

### 不合理 / 需关注之处

1. **化学正式入门总时长仍偏紧**：4 次 × 30min = 2h，对零基础学生只能建立粗浅框架，开学后课堂节奏可能跟不上。
2. **8-10min 背 2-3 个方程式 + 元素周期表前 5 号可能偏紧**：若要求孩子完整书写（含配平、条件、↑/↓），2 个方程式的质量优于 3 个方程式的赶工。
3. **方程式背诵与元素周期表滚动叠加**：元素周期表前 5 号（H、He、Li、Be、B）较简单，但后续加入前 20 号时，需预留更多时间。
4. **化学 30min 课程依赖孩子自学或视频**：若无教师讲解，零基础学生理解化学式、化合价、配平可能困难。

### 化学教师建议

- **将方程式目标调整为「每天 2 个，确保完整正确」**，质量优先；学有余力再背第 3 个。
- **每次 30min 化学课前，要求孩子先花 10min 预习课本对应章节**，标注不懂的名词，提高课堂/视频效率。
- **提供「暑假必背 30 个化学方程式」清单**，按周分解，避免临时选方程式的随意性。
- **若 8.10 后发现理解困难，可在周六上午增加 1 次 30min 化学答疑/视频回看**（需同步调整周末表，但 Saturday 全天休息原则仍可保留下午/晚上）。

## 七、历史 / 道法教师视角

### 合理之处

1. **隔天 20min 不影响主科**，历史/道法分工明确，与「历史保温、道法补漏」定位一致。
2. **道法日必须输出 1 个「是什么-为什么-怎么做」答题模板**，有输出要求，避免流于形式。
3. **周日 20:20-20:30 增加 10min 道法索引整理**，在不增加工作日负担的前提下略增道法权重。
4. **8.28 晚历史道法合卷感知 30min**，让学生提前体验合卷卷面、时间分配和答题顺序。
5. 打卡表历史日「画时间轴/说 3 个关键词/复述 1 个因果链」、道法日「贴便签索引/口述模板」均可验收。

### 不合理 / 需关注之处

1. **道法 45 分（年级第 51）与历史 65 分（年级第 5）差距大**，但除周日 10min 外，两科时间仍均分；道法提分空间大，应更倾斜。
2. **20min 容易被主科挤占**：若当天其他任务超时，历史/道法是最可能被牺牲的科目。
3. **合卷感知仅 1 次**：30min 浏览卷面只能建立初步印象，不足以形成稳定的答题策略。
4. **道法模板输出需要家长判断质量**：家长可能分不清「是什么-为什么-怎么做」是否完整。

### 历史 / 道法教师建议

- **将周日 10min 道法索引整理扩展为 15-20min**，或增加周六晚名著轮读后 10min 道法模板回顾。
- **为家长提供 3-5 个道法答题模板示例**，验收时让孩子口述并对比示例即可。
- **8.28 合卷感知后，8.29 或 8.30 再安排 1 次 20min 合卷选择题限时训练**，巩固时间分配。
- **历史重点建立世界史时间轴**，道法重点记忆三类答题模板，不追求暑假大幅提升，但求框架清晰。
