# 每日打卡表 · v2

> 周期：2026.7.13 - 2026.8.31
> 模式：周一/三/五 = A日，周二/四 = B日
> 周末：周六上午休、下午化学、晚上自由活动；周日白天休、晚上预备
> 中考体育：一类 4 分钟跳绳 + 二类 1 分钟速跳 + 三类排球
> 来源：[AI:Claude]

## 使用说明

1. **语文古诗 9:00-9:30**：按表背诵/默写，用豆包语音抽查。
2. **作文素材/模板 16:30-16:45**（B日为主）：积累作文素材或阅读答题模板。
3. **英语 9:40-10:10 + 16:45-17:30**：A日新学单元词+专项训练；B日错词复习+语法专项。
4. **体育训练**：周一/三晚饭前跳绳，周五晚饭后跑步，周末排球。
5. **豆包验收指令**：
   - 语文：`请听我背诵《XXX》，抽查 5 个重点字词，让我默写 2 句。`
   - 英语：`请给我听写 XX 个核心单词和 5 个短语，错词让我再默写一遍。`
6. 完成当天任务后在 **完成** 栏打勾。

## 每日豆包验收指令模板

### 语文古诗背诵验收
```
我是初二升初三学生，今天背诵任务如下：
[粘贴当天语文任务]

请你：
1. 听我背诵（我可以语音输入），指出错误；
2. 抽查 5 个重点字词的意思；
3. 让我默写其中最容易错的 2 句。
```

### 英语听写验收
```
我是初二升初三学生，使用沪教牛津版英语。
今天复习 [七上/七下/八上/八下/九上] Unit X，
请你给我听写 15 个核心单词和 5 个重点短语。
听写完后帮我批改，并把错词加入错词本，最后让我再默写一遍错词。
```

### 英语错词复习验收
```
请帮我复习上周英语错词本中的 20 个单词/短语，
先让我自己拼写，然后批改，最后再让我把错词重新默写一遍。
```
