### 2. 英语 · 重点提升（基础补强）

**现状：** 基础薄弱，暑假以"补基础"为核心，不盲目追进度。

**核心任务：**
- 四年级全册单元单词：过关听写、拼写、释义
- 四年级课文背诵：Let's talk / Let's learn 对话熟练朗读并背诵
- 同步学内容：跟紧五年级上册预习，重点词汇与句型先行接触

**基础补强（针对薄弱环节）：**
- 字母与书写：26个字母规范书写 + 四线三格
- 自然拼读（Phonics）入门：常见字母组合发音规律
- 高频词（Sight Words）：常见220词认读
- 常用句型：问候、介绍、喜好、时间、天气等日常表达

**学习方式（尽量非电子）：**
- 纸质单词卡片随身记
- 课文大声朗读 + 背诵打卡
- 每日手写练习：抄写、听写、造句
- 限时听力：每日1次，每次≤15分钟（仅用于发音校准）

**建议节奏：**
- 每日英语总时长：60-90分钟（分2-3段，避免疲劳）
- 单词：每日新词10-15个 + 滚动复习旧词
- 课文：2-3天背1篇

### 3. 语文 · 基础夯实

**现状：** 语文同样需夯实基础，不盲目拔高，先把字词、朗读、书写、基础表达打牢。

**核心任务：**
- 四年级生字词全面过关：听写、形近字辨析、多音字、词语搭配
- 课文朗读：正确、流利、有感情，重点课文熟读成诵
- 古诗文：五年级上册必背古诗诵读背诵（重积累，不强求赏析）
- 写字：硬笔书法，规范端正，每日15-20分钟
- 基础写作：每周2-3篇日记/短文，把话写清楚、写通顺
- 课外阅读：每日30-45分钟，培养习惯与基础理解力（纸质书）

**学习方式（非电子优先）：**
- 字词听写本 + 错题本（纸质）
- 大声朗读课文，妈妈抽查背诵
- 手写日记，老师/妈妈点评

## 四、其他安排（按年龄与小学最佳实践）

针对10-11岁儿童，结合"减少室内与电子产品"原则：

- 上午：跳绳、跑步、球类

- 中文阅读为主，英文绘本/分级读物为辅

### 3. 劳动与生活技能
- 每日家务：整理房间、洗碗、简单烹饪
- 培养独立自理能力

### 4. 兴趣与美育
- 绘画、手工、乐器（非电子类优先）
- 每周1-2次

### 5. 社交与体验
- 每周1次户外活动/博物馆/自然探索
- 与同伴互动

### 6. 作息与电子产品
- 睡眠：21:30前入睡，保证9.5-10小时
- 电子产品：每日累计≤30分钟（仅限必要学习/听力）
- 用眼：每20分钟远眺20秒

## 六、注意事项

1. **语英基础为先**：英语与语文均需夯实基础，优先保证两科每日时长，不因其他安排挤占；英语为最薄弱项，时长优先级最高
2. **减少室内与电子产品**：学习多用纸质，听力限时，娱乐以户外为主
3. **劳逸结合**：5日1休的节奏同样适用于日常，每周安排1天"放松日"
4. **家校协同**：妈妈与各科老师保持沟通，每周小结1次，动态调整
5. **关注情绪**：避免过度安排，保护学习兴趣与亲子关系
