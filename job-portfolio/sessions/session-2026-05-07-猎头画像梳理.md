# 2026-05-07 对话记录 — 猎头画像梳理（资深猎头引导模式）

## 面试口径（最终版）

> "我在现公司将近4年，从0到1搭建了技术体系和研发团队，经历了公司从A轮到业务成熟的完整周期。近期公司业务方向有调整，研发资源会进行结构性优化，管理的纵深和复杂度随之降低，技术方向的落地空间也相应收窄。同时，我2025年以来一直在深度实践AI编程和Vibe Coding，也推动团队在这方面做了不少尝试，但受限于公司当前阶段的战略重心，这方面的持续投入空间有限。所以我希望找到一个技术驱动、愿意在AI工程化方向持续投入的平台，把我这些年的技术管理经验和AI实践结合起来，创造更大的价值。"

**口径设计逻辑**：
1. **"将近4年"** → 用时间信用背书，不是"刚入职就跑路"
2. **"从0到1搭建"** → 强调 ownership 和创造感，不是"接手现成的"
3. **"A轮到业务成熟"** → 暗示完整周期已走完，离开是自然节点
4. **"结构性优化"** → 用中性词替代"裁员"，但已被验证在追问下会暴露（见5月8日面试演练Q1反馈）
5. **"管理纵深降低"** → 不抱怨、不情绪化，用岗位价值维度解释离开原因
6. **"AI编程+Vibe Coding"** → 展示技术前瞻性，与目标岗位（技术驱动）对齐
7. **为什么不说"被优化"** → 被动标签会丢掉 leverage；不说"求稳" → 暴露安全诉求
8. **为什么不说"薪资不满意"** → 猎头面试阶段谈钱=自降身份，等对方来问

### 关键洞察
- 核心恐惧："被当成一次性工具人"（现公司A轮→业务成熟→结构性优化）
- 核心诉求：找"认可技术管理层长期价值"的平台
- 面试反向考察3问（测CEO是否把你当合伙人）：
  1. "公司对技术团队的定位是什么？是业务支撑部门，还是核心竞争壁垒？"
  2. "技术VP/总监这个岗位，3年后的成功画像是什么？"
  3. "公司下一轮融资或者盈利 timeline 是什么？技术投入在其中的角色？"

**推导过程**：
- **核心恐惧的来源**：现公司经历是"A轮招人搭体系→业务成熟→结构性优化"，本质是"技术管理层被用完即弃"
- **核心恐惧的验证**：5月8日面试演练Q3中，CEO问"你会不会搭完就跑"——这个问题恰恰说明对方也有"你是不是用完即弃"的担忧，你的恐惧和对方是对称的
- **反向考察3问的设计逻辑**：
  - Q1测**技术定位**：如果对方说"业务支撑"=你来了就是打杂的，随时可替换；说"核心竞争壁垒"=技术投入有保障
  - Q2测**长期规划**：3年后的成功画像=公司对这个岗位的期待是"长期合伙人"还是"短期执行者"
  - Q3测**资金确定性**：融资/盈利 timeline + 技术投入角色=验证"技术投入是否有长期确定性"（与面试口径中"技术投入有长期确定性"呼应）
- **3问的使用时机**：不宜在初筛/HR面抛出（对方没决策权），应在CTO/CEO终面最后阶段提出，作为"双向选择"的信号

### 薪资package（2026-05-07 补充）
- **现有总包**：100万/年
  - 期权：30万/年（名义），已缴纳25%现金，公司未盈利无法兑现/分红，离职返还本金+年化5%利息
  - 现金：70万/年，结构为80%月薪 + 20%绩效
  - 福利：每月500补贴 + 每日2餐 + 保险
- **期望package**：现金不低于70万（底线80万更佳），总包有真实涨幅；长期激励需有明确兑现路径
- **谈判核心原则**：强调"确定性"而非"金额"，不暴露现有期权的弱势条款

**修正推导**：
- 最初画像写"50-100万"→ 范围太宽，猎头会按50万推岗，浪费双方时间
- 修正为"现金70万+"的原因：
  1. 现有现金就是70万，低于这个数=降薪跳槽，无动力
  2. 底线80万更佳：给谈判留空间，也筛选掉预算不足的公司
  3. 总包需"真实涨幅"：因为现有期权≈0（未盈利、离职返还本金+5%利息），所以"总包100万"是虚的，真实现金只有70万
- **为什么不暴露期权弱势条款**：一旦对方知道你的期权是"名义上的"，会大幅压低现金报价（"反正你现在的package也没多少"）
- **强调"确定性"的策略**：成长期公司期权多现金少是常态，强调"确定性"=暗示"我吃过期权的亏"，引导对方在现金上做让步
