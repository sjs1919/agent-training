## §3 错误响应

### 错误响应格式

```json
{"code": 40001, "message": "<中文错误描述>", "data": null}
```

### 错误码编码规则

5 位码 = 3 位 HTTP 状态码 + 2 位业务序号。例：`40001` = HTTP 400 + 业务序号 01。

| 前缀 | 类别 |
|------|------|
| `400xx` | 参数/请求错误 |
| `401xx` | 认证失败 |
| `402xx` | 余额/支付 |
| `403xx` | 权限不足 |
| `404xx` | 资源不存在 |
| `409xx` | 状态冲突 |
| `429xx` | 限流 |
| `500xx` | 服务端错误 |
| `502xx` | 上游 API 错误 |
